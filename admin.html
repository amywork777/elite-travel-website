Admin Guide

How to manage tours on your website

1 Add a New Tour

  • Click the "+ Add New Tour" button at the top
  • Fill in the Basics tab: title (English & Chinese), destination, duration, price, dates
  • The slug is auto-generated from the title — it becomes the URL (e.g. /tour/japan-cherry-blossom)
  • Add a cover image by clicking "Choose File" — this is the main photo shown on cards
  • Fill in the other tabs: Details, Itinerary, Includes/Excludes
  • Click Save when done

2 Make a Tour Visible

  • New tours start as Draft (not visible to the public)
  • To publish: edit the tour → go to the Settings tab → change status to Live
  • To feature on the homepage: check the "Featured" box (up to 6 featured tours will show)
  • To hide a tour: change status back to Draft

3 Edit or Delete a Tour

  • Click Edit on any tour card to modify it
  • Click Delete to permanently remove a tour (this cannot be undone)
  • Changes take effect immediately on the website

4 Bilingual Content

  • Each text field has an English and Chinese version
  • If you leave a Chinese field empty, the English version will be shown to Chinese visitors
  • Visitors can switch languages using the EN/中文 toggle on the website

5 Import Multiple Tours (CSV)

  • Click "Import CSV" to add many tours at once
  • Prepare a CSV file with columns: title, slug, destination, duration, price_from, summary, description
  • You can also include: title_zh, destination_zh, summary_zh, description_zh for Chinese text
  • Upload the file → preview the data → click "Import All"

Tip: Always fill in both English and Chinese titles for the best experience. The cover image should be landscape format (wider than tall) for best results.
